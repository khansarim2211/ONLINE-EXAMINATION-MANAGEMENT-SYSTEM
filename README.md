# EXAMIFY – Online Examination Management System

EXAMIFY is a secure and user-friendly web-based Online Examination Management System developed using Django.

## Features
- Secure Login Authentication
- Random Question Generation
- Online Exam Interface
- Automatic Result Evaluation
- Student Performance Tracking
- Responsive User Interface

## Technologies Used
- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite

## Modules
- Student Module
- Teacher Module
- Result Management
- Authentication System

## Future Scope
- AI-based analytics
- Cloud deployment
- Mobile support
